##  **BASES DE DONNÉES UTILISÉES**

### **1. PROJET CANCER DU SEIN**
**Source :** Breast Cancer Wisconsin Dataset (Diagnostic)
- **Origine :** Université du Wisconsin, USA
- **Contributeurs :** Dr. William H. Wolberg et al.
- **Date de création :** 1995
- **Accès :** Public, inclus dans scikit-learn

**Caractéristiques :**
- 569 échantillons (212 malins, 357 bénins)
- 30 caractéristiques par échantillon
- Features calculées à partir d'images de noyaux cellulaires :
  - Rayon (moyen des distances centre-contour)
  - Texture (écart-type des niveaux de gris)
  - Périmètre, Aire, Compacité, Concavité, Points concaves
  - Symétrie, Dimension fractale

**Méthode d'acquisition :**
- Aspiration à l'aiguille fine de masses mammaires
- Numérisation microscopique
- Segmentation automatique des noyaux
- Calcul de caractéristiques morphologiques

### **2. PROJET TUMEURS CÉRÉBRALES**
**Source :** Brain MRI Images for Brain Tumor Detection
- **Plateforme :** Kaggle
- **Contributeur :** Navoneel Chakrabarty
- **Date de publication :** 2020
- **Licence :** CC0: Public Domain

**Caractéristiques :**
- 253 images IRM cérébrales
- Répartition :
  - Classe 'yes' (avec tumeur) : 155 images (61.3%)
  - Classe 'no' (sans tumeur) : 98 images (38.7%)
- Format : Images JPEG, résolution variable
- Modalité : IRM pondérée en T1/T2

**Origine des images :**
- Collecte rétrospective de scans patients
- Annotations validées par radiologues
- Désidentification conforme HIPAA

##  **MÉTHODOLOGIE EXPÉRIMENTALE DÉTAILLÉE**

### **A. PROJET 1 : PIPELINE RANDOM FOREST**

```python
# Workflow scientifique complet
1. Chargement dataset Wisconsin depuis scikit-learn
2. Simulation réaliste : injection de 5% de NaN
3. Imputation par moyenne (avec avertissement data leakage)
4. Split stratifié 80/20 (random_state=42 pour reproductibilité)
5. Entraînement RandomForestClassifier (n_estimators=100)
6. Évaluation multicritère : Accuracy, Precision, Recall, F1
7. Visualisation : Matrice de confusion, importance des features
```

**Choix algorithmique justifié :**
- Random Forest : robuste à la multicollinéarité
- Bagging + Feature Randomness : réduction de variance
- Pas de normalisation nécessaire (algorithmes arbres)

### **B. PROJET 2 : PIPELINE CNN**

```python
# Workflow Deep Learning médical
1. Chargement images depuis répertoire Kaggle
2. Prétraitement :
   - Redimensionnement à 128×128
   - Normalisation [0,1] (÷255)
   - Augmentation de données (rotation, zoom, flip)
3. Split triple : 70% train, 15% val, 15% test
4. Architecture CNN personnalisée :
   - 3 blocs Conv2D (32→64→128 filtres)
   - BatchNormalization + MaxPooling après chaque bloc
   - Dropout (25-50%) pour régularisation
   - Sortie sigmoid pour classification binaire
5. Compilation : Adam optimizer (lr=0.001), Binary Crossentropy
6. Entraînement avec EarlyStopping (patience=10)
7. Évaluation exhaustive sur test set
```
